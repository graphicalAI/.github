# graphicalAI

Добро пожаловать в организацию **graphicalAI**! Мы занимаемся разработкой интеллектуальных систем визуализации и конструирования нейронных сетей.

## 🚀 Наш главный проект
*   **[graphNet](https://github.com/graphicalAI/graphNet)** — кросс-платформенный визуальный конструктор нейросетевых архитектур в виде графов.

## 📋 Программа стажировки и разработки
Мы активно сотрудничаем с талантливыми разработчиками и стажерами. В нашей организации действуют строгие регламенты:
1.  **Процесс разработки**: вся работа ведется через Pull Requests. Коммиты в `main` напрямую закрыты. Изменения сначала вносятся в `develop`, тестируются и только потом попадают в релиз.
2.  **Задачи и доски**: Все текущие задачи находятся на доске проектов **[GraphNet Project](https://github.com/orgs/graphicalAI/projects/3)**.
3.  **Правила оформления**: Перед началом работы ознакомьтесь с [CONTRIBUTING.md](./CONTRIBUTING.md).

## 🎓 Развитие стартапа и грант ФСИ
Проект развивается при поддержке Фонда содействия инновациям (ФСИ) в рамках программы «Студенческий стартап». 
*   **Внимание для стажеров ГПХ**: формулировки решаемых задач (Issues) должны строго соответствовать этапам Календарного плана гранта! Не используйте абстрактные заголовки при создании задач.
