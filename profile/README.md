# graphicalAI

Добро пожаловать в организацию **graphicalAI**! 

Наша главная цель — создать полноценный графический конструктор нейронных сетей, не уступающий по своим функциональным возможностям профессиональным фреймворкам вроде **Keras** и **TensorFlow**. Мы стремимся снизить порог входа в сферу глубокого обучения и предоставить разработчикам мощный визуальный инструмент для проектирования и прототипирования сложных моделей. Наш флагманский продукт — **graphNet** — доказывает, что графическое проектирование нейросетей может быть столь же гибким и производительным, как и традиционное написание кода.

## 🛠 Наши проекты

*   **[graphNet](https://github.com/graphicalAI/graphNet)** — кросс-платформенное десктопное приложение для визуального конструирования архитектур нейронных сетей на базе Dear PyGui. Позволяет интерактивно собирать архитектуру, настраивать гиперпараметры слоев и визуализировать потоки данных.

Мы регулярно привлекаем увлеченных разработчиков в нашу команду. Если вы хотите пройти стажировку в graphicalAI, пожалуйста, свяжитесь напрямую с основателем проекта Георгием в Telegram: **[@jezvGG](https://t.me/jezvGG)**.

### 📋 Процесс отбора состоит из 2 этапов:
1.  **Собеседование** — знакомство, обсуждение вашего опыта и целей.
2.  **Тестовое задание** — решение небольшой реальной задачи (Issue) из нашего репозитория.

### ⚙ Регламент для участников команды и стажеров:
*   **Git-процесс**: Все изменения вносятся строго через Pull Requests в ветку `develop`. Прямые коммиты в `main` заблокированы.
*   **Управление задачами**: Все актуальные задачи ведутся на доске проектов **[GraphNet Project](https://github.com/orgs/graphicalAI/projects/3)**. Перед началом работы ознакомьтесь с [CONTRIBUTING.md](./CONTRIBUTING.md).
